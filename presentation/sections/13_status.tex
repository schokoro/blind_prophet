      \textbf{\small Три границы метода}

      \vspace{0.4em}
      \begin{tcolorbox}[
          colback=orange!8, colframe=orange!50,
          boxrule=0.4pt, arc=3pt,

      ]
        \scriptsize
        Агрессивная нейтрализация ($\Delta=0.88$) убрала числовые якоря,
        нужные для калибровки. Neutered хуже raw.\\
        \textbf{Сильнее ≠ лучше.}
      \end{tcolorbox}

      \vspace{0.3em}
      \begin{tcolorbox}[
          colback=red!7, colframe=red!40,
          boxrule=0.4pt, arc=3pt,
          left=5pt, right=5pt, top=3pt, bottom=3pt,
          fonttitle=\footnotesize\bfseries, title={2.~Empirical ceiling · 2022-03}
      ]
        \scriptsize
        Лексические маркеры убраны, но уникальная макроконфигурация
        (FX + ставка + цепочки) всё равно идентифицирует период.\\
        \textbf{Engelberg ceiling на реальных данных.}
      \end{tcolorbox}

      \vspace{0.3em}
      \begin{tcolorbox}[
          colback=gray!8, colframe=gray!40,
          boxrule=0.4pt, arc=3pt,
          left=5pt, right=5pt, top=3pt, bottom=3pt,
          fonttitle=\footnotesize\bfseries, title={3.~Recovery dynamics · 2022-04/05/06}
      ]
        \scriptsize
        Население адаптировалось быстро, новости остались плохими.
        Метод читает медиатон, не читает адаптацию.\\
        \textbf{Overshoot устойчив на 3 точках.}
      \end{tcolorbox}

    \end{column}

    %% ---- Right column: known limitations + directions ----------------
    \begin{column}{0.44\textwidth}

      \textbf{\small Известные ограничения PoC}

      \vspace{0.3em}
      \begin{itemize}\scriptsize\setlength\itemsep{2pt}
        \item[$\cdot$] Стохастичность цикла — нужно $N{=}K$ прогонов
        \item[$\cdot$] Blacklist слишком жёсткий на родовых словах эпохи
        \item[$\cdot$] $n{=}10$ точек, формальная статистика невозможна
      \end{itemize}

      \vspace{0.8em}
      \textbf{\small Направления}

      \vspace{0.3em}
      \begin{itemize}\scriptsize\setlength\itemsep{2pt}
        \item[$\rightarrow$] $N{=}K$ reruns → характеристика stability
        \item[$\rightarrow$] J2-grounded blacklist (data-driven)
        \item[$\rightarrow$] Серия 36–48 точек + significance test
        \item[$\rightarrow$] Persona ablation study
        \item[$\rightarrow$] Отдельные модели Summarizer / Neutralizer
      \end{itemize}

      \vspace{0.8em}

        neutered IQR широкий (каждая персона — свой prior).
      \end{tcolorbox}
